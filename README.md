Iris
====

A programming language

## Introduction

### Variables

```iris
let varname value
let varname :anothervarname
```

### Printing

```iris
print hello world
print :varname
print %ln
print %retval
```

### Arithmetic

```iris
add 1 2
print 1 + 1 = %retval
```

### More Arithmetic

```iris
div 5 2 2
print %retval
```
