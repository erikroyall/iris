Iris
====

A programming language

## Introduction

### Variables

```
let varname value
let varname :anothervarname
```

### Printing

```
print hello world
print :varname
print %ln
print %retval
```

### Arithmetic

```
add 1 2
print 1 + 1 = %retval
```

### More Arithmetic

```
div 5 2 2
print %retval
```

### Labels

```
goto main

this too

main:

```

### if statement

```
if 2 < 3 print hello
```

### More examples

